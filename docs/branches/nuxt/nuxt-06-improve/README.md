À la fin de cette branche, vous serez capable de :

- ✅ Utiliser le scraping pour enrichir la base de données

- ✅ Ajouter la route `POST /api/offers/import` dans `server/api/offers/import.post.ts`
- ✅ Parser le HTML de stage.fr et extraire le titre de chaque annonce dans un premier temps
- ✅ Mapper les données offres v1
- ✅ Persister les offres dans json-server
- ✅ Dans un second temps, ajouter la persistance de la description, de la location et du lien de l'offre
